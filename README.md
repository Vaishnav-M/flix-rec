# Flix Rec — Netflix-like React UI

This is a minimal React + Vite frontend that mimics a Netflix-style UI: navbar, hero banner, and horizontal rows of movie posters.

Quick start:

```bash
npm install
npm run dev
```

Open http://localhost:5173

Notes:
- Uses static sample data in `src/data/movies.js` (picsum placeholders).
- Styling is plain CSS in `src/index.css`.
